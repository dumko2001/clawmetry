ClawMetry for Mac

Monitor your OpenClaw AI agents from the menu bar. Native Swift app. No browser tab needed.

Download for macOS

v1.0.1 · macOS 13+ · View source on GitHub

📊

Menu Bar Access

Lives in your menu bar. Click to open the full dashboard. No dock icon, no browser tab. Always one click away.

🔒

E2E Encrypted

Your data is encrypted on your machine before it leaves. Decrypted in the app on demand. Nothing reaches the cloud in plaintext.

⚡

Persistent Sessions

No more login loops. Auth tokens survive restarts. Session storage backed by localStorage. Reload throttling built in.

🔄

Live Flow View

Watch your agents work in real-time. Animated particle flow between gateway, sessions, tools, and sub-agents.

💰

Token Cost Tracking

See exactly what your AI agents cost. Per-session, per-model breakdowns. Set budgets and get alerts.

🧠

Memory Browser

Browse your agent's memory files directly. See what it remembers, what it forgot, and what changed.

Built with Swift + WKWebView

Native macOS app, not an Electron wrapper. Uses WKWebView with persistent data store, shared process pool, and intelligent reload throttling.

Open source. 338 lines of Swift. Read the code.
