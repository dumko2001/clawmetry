Strategy Guide

How to use these decks

All four versions now tell the bigger story: ClawMetry is the universal observability layer for AI agents. OpenClaw is the wedge, not the ceiling. Choose the angle that resonates with each investor.

🎯 Default / First Meeting

Start with V2 (Universal Platform) for most investors. It tells the biggest story: 100-agent companies need observability, and ClawMetry is building it. Use the Main Deck (pitch-deck.html) for the cleanest version without version tags.

📈 Metrics-First Investors

Lead with V4 (Growth & Metrics) for spreadsheet VCs. $0 CAC, 85% margins, and the TAM expansion math (each framework multiplies the market) are the hooks.

🤖 Technical Angels / CTOs

Lead with V3 (Product Depth). 7 tabs, zero-knowledge encryption, plugin architecture, auto-detection engine. The technical moat across multiple frameworks is the selling point.

🏢 OSS / Ecosystem VCs

Lead with V1 (OpenClaw Wedge). Shows the proven beachhead (326k stars, 75k installs) and the clear expansion path through the Claw ecosystem to universal coverage.

Narrative Shift

What changed in this update

❌ Old Positioning

"Datadog for OpenClaw" -- treated ClawMetry as a tool for one framework. Limited TAM. Single-framework dependency risk.

✔ New Positioning

"The universal observability layer for the AI agent era" -- OpenClaw is the wedge, not the ceiling. TAM is $12B+ (all agent users), not $50M (OpenClaw users). Platform play, not tool play.

Key additions across all decks

NemoClaw (NVIDIA), NanoClaw (24k stars), PicoClaw (25k stars) ecosystem. Auto-detection vision. Non-technical friendly positioning. "100 agents per company" framing. TAM expansion math per framework.

Revenue story upgrade

$5/node/month scales in two dimensions: conversion rate AND framework expansion. More agents = more nodes = more revenue per customer. Built-in expansion revenue. 130%+ NRR projected.
